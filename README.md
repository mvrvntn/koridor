# коридор

Минималистичный сервис для навигации в цифровом пространстве.

## О проекте

**коридор** — это минималистичный веб-сервис, созданный в стиле студии Art. Lebedev. Проект сочетает в себе строгость дизайна, функциональную простоту и современную веб-технологию.

### Особенности

- Минималистичный дизайн в стиле Art. Lebedev
- Плавные CSS-анимации и переходы
- Полностью адаптивная верстка с использованием CSS Grid и Flexbox
- Оптимизированная производительность загрузки
- Семантическая HTML5-разметка
- Поддержка всех современных браузеров и устройств

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript
- Google Fonts (Inter)

## Разработка

Проект готов к развертыванию на GitHub Pages. Все ресурсы оптимизированы для быстрой загрузки и соответствуют современным веб-стандартам.

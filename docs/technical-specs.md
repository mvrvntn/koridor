# Технические спецификации проекта "Коридор"

## Обзор проекта

"Коридор" - это интерактивная веб-страница с анимированным маскотом-курсором и эффектом перехода "туннель". Проект представляет собой технологический сервис с минималистичным дизайном и уникальным визуальным стилем.

## Архитектура и технология

### Основной стек
- **HTML5**: Семантическая вёрстка с современными стандартами
- **CSS3**: Модульная организация стилей с переменными CSS
- **Vanilla JavaScript**: Оптимизированный JavaScript без зависимостей
- **GSAP**: Библиотека анимаций для высокопроизводительных эффектов
- **SVG**: Векторная графика для масштабирования и иконок

### Файловая структура
```
koridor/
├── index.html          # Основной HTML-файл
├── assets/
│   ├── css/
│   │   └── style.css      # Основные стили
│   ├── js/
│   │   └── main.js       # Основной JavaScript
│   └── images/              # Изображения и иконки
├── docs/
│   ├── technical-specs.md # Технические спецификации
│   └── deployment.md        # Инструкции по развертыванию
├── .nojekyll                 # Отключение Jekyll для GitHub Pages
├── .gitignore               # Исключения файлов Git
├── LICENSE                  # Лицензия MIT
├── 404.html               # Страница ошибки 404
└── sitemap.xml             # Карта сайта для SEO
```

## Требования к производительности

### Целевые показатели
- **First Contentful Paint (FCP)**: < 1.5с для текста и фона
- **Largest Contentful Paint (LCP)**: < 2.5с для интерактивных элементов
- **Speed Index**: > 90 для мобильных устройств
- **Time to Interactive**: < 100мс для первого взаимодействия

### Оптимизации
- **Критические пути рендеринга**: Оптимизация последовательности отрисовки элементов
- **Ленивая загрузка**: Отложенная загрузка некритичных ресурсов
- **Аппаратное ускорение**: Использование GPU для анимаций
- **Оптимизация памяти**: Эффективное управление объектами JavaScript

## Требования к доступности

### Уровень соответствия
- **WCAG 2.1 AA**: Полное соответствие стандартам доступности

### Ключевые элементы
- **Клавиатурная навигация**: Полная поддержка клавиатурной навигации
- **Экранные считыватели**: Совместимость с программами чтения с экрана
- **Контрастность**: Соответствие коэффициенту контрастности 4.5:1
- **Адаптивный текст**: Масштабируемый текст без потери качества

## Требования к SEO

### Метаданные
- **Open Graph**: Полные метаданные для социальных сетей
- **Twitter Cards**: Оптимизированные карточки для Twitter
- **JSON-LD**: Структурированные данные для поисковых систем
- **Микроразметка**: Schema.org для улучшения понимения контента

## Требования к безопасности
